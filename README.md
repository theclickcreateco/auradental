# Cosmetic Dental Studio — Branding Portfolio

A high-end, responsive landing page demonstration for **Aura Dental Collective**. This project showcases the "Ethereal Clinic" design system, focusing on soft minimalism, glassmorphism, and clinical precision.

## 📂 Project Structure

- [index.html](file:///d:/Cinorium/Repos/Portfolio/landingpages/cosmetic-dental-studio/stitch/dental_studio_landing_page_with_booking_popup/index.html): Unified landing page core.

## ✨ Key Features

- **Ethereal Design System**: Custom tonal palette, 20px glassmorphism, and high-gloss UI elements.
- **Interactive Transformations**: Fully functional Before/After smile slider.
- **Modular Bento Gallery**: A refined visual showcase of dental artistry.
- **Fully Responsive**: Optimized for mobile, tablet, and desktop viewports with zero horizontal overflow.
- **PWA Ready**: Offline caching and home-screen installation via Service Workers.
- **Direct Lead Capture**: Integrated booking modal with demonstration lead routing.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla JavaScript.
- **Styling**: Tailwind CSS (via CDN).
- **Icons**: Material Symbols (Variable).
- **Branding**: Custom iconography generated for Aura Dental Collective.
